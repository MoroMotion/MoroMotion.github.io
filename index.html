Introducing GLENN

Gait Learning and Evaluation using Neural Networking

Technical Information

GLENN uses a mix of synchronous and servo motors to control his limbs. Re-chargeable lithium-ion batteries supply 24 Volts to GLENN’s system which powers 10+ synchronous cycloidal drive motors for his arms, legs, and core. A LattePanda 3 Delta computer with an external AI processor controls the entire robot system.

AI Implementation

GLENN was trained using NVIDIA’s Isaac Lab/Sim, a simulation and Reinforcement Learning (RL) training software that uses a complex Proximal Policy Optimization (PPO) algorithm to run thousands of iterations that help GLENN learn how to navigate through real world landscape. This trained policy is deployed onto the LattePanda 3 Delta which combined with the AI Accelerator create a bridge between the trained neural network and the hardware.
